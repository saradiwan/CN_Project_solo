Validation of IP Address

An IP address (Internet Protocol address) is a unique numerical identifier assigned to each device connected to a network. It enables communication between devices over the internet or within a local network.

IPv4 addresses contain four numbers separated by dots. Each number must be between 0 and 255. If any value goes outside this range, the IP address is considered invalid.

Examples of valid IP addresses include 192.168.0.1 and 10.0.0.5. This project validates whether the entered IP address follows the correct format.
